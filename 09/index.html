最大公約数

以下の仕様を満たすように、関数を定義してください。定義は 09.js に記述してください。

記法


let number = gcd(naturalA, naturalB);

引数

  • naturalA：最大公約数の計算に用いる 1 つ目の自然数
  • naturalB：最大公約数の計算に用いる 2 つ目の自然数

返り値

引数に指定された整数の最大公約数

利用例


let a, b;
a = 128;
b = 76;
let c = gcd(a, b);
console.log(`gcd(${a}, ${b}) = ${c}`);
