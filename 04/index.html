論理演算

以下の仕様を満たすように、論理演算を行う関数 xor を定義してください。定義は 04.js に記述してください。

記法


let result = xor(boolA, boolB);

引数

  • boolA：1つ目の真偽値
  • boolB：2つめの真偽値

返り値

引数の値に応じて、以下の値を返します。

第1引数第2引数返り値
truetruefalse
truefalsetrue
falsetruetrue
falsefalsefalse

利用例


xor(true, true);   // false
xor(false, true);  // true
xor(true, false);  // true
xor(false, false); // false
