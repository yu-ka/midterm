解答するための事前準備

中間試験に解答するためには、次の準備を済ませておく必要があります。準備はこれまでの授業で全て済ませているはずですが、済ませていない場合は、解答する前に全て設定しておいてください。

  • Firefox Developer Edition のインストール
  • GitHub Desktop のインストール
  • Visual Studio Code といった、エディタ / IDE の用意
  • GitHub へのアカウント登録

特に GitHub のアカウント登録は、解答の提出のために必須となっています。登録されていない方は、こちらの登録ページからアカウント登録をしてください。

また登録されたアカウントを、清水にお知らせいただけないと、採点結果を成績に反映できなくなります。新たに GitHub のアカウント登録をされた方、もしくはこれまでにお知らせいただいていない方は、こちらのフォームにご記入ください。

解答方法

  • 試験問題は、01 から 10 までのフォルダに分かれています。
  • それぞれのフォルダ内の index.html を Firefox で開き、そこに書かれている要件を満たす関数を定義してください。
  • 関数の定義は、問題と同じフォルダにある js ファイルに記述してください。
  • 要件の充足率が、試験の点数となります。
  • 関数定義後、問題の記述されているページを再読み込みすることで、要件の充足度が測れます。

提出方法

  • 提出は GitHub で pull request を送ることによって行います。
  • pull request の送り方は、こちらの文書をご覧ください。
  • pull request が送られていない場合、受験されなかったものとして扱います。
