恒等写像

引数に指定された値を、そのまま返す関数 id を 01.js に定義してください。

記法 

let result = id(value);

引数

  • value：なんらかの値

返り値

引数に指定された値

利用例


let result = id(1);
result == 1; // true;

let text = id("hello");
text == "hello"; // true
