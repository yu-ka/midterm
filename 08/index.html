うるう年判定

以下の仕様を満たすように、関数を定義してください。定義は 08.js に記述してください。

記法


let result = isLeapYear(yearInAD)

引数

  • yearInAd：紀元後の年を表す数字（0以上の整数）

返り値

引数に指定された数値に対してグレゴリオ暦に基づくうるう年判定を行い、うるう年の場合は true を、そうでない場合は false を返す

利用例


isLeapYear(2000); // true
isLeapYear(2001); // false
isLeapYear(2004); // true
ieLeapYear(1900); // false
